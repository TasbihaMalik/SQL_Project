# 📖 About This Repository

This repository is a comprehensive collection of **SQL practice exercises** designed to help students, beginners, and aspiring Data Analysts build a strong foundation in SQL.

Each SQL file focuses on a specific topic and contains practical examples that simulate real-world database operations. The exercises gradually progress from basic SQL syntax to more advanced concepts such as Views, Foreign Keys, Subqueries, Aggregate Functions, UNION operations, and Stored Procedures.

Whether you're preparing for interviews, improving your SQL skills, or learning databases for the first time, this repository provides structured practice to strengthen your understanding of SQL.

# 🚀 Getting Started

## 1️⃣ Clone the Repository

```bash
git clone https://github.com/TasbihaMalik/SQL-Practice.git
```

---

## 2️⃣ Create Database

```sql
CREATE DATABASE sql_practice;
```

---

## 3️⃣ Select Database

```sql
USE sql_practice;
```

---

## 4️⃣ Execute SQL Files

Run each SQL file individually.

Example:

```sql
SOURCE Sql_Basic_Practice.sql;
```

or

Open the SQL file in **MySQL Workbench** and click **Execute**.
